# 日期分隔符UI优化总结

## 修改概述
重新设计时间轴列表中的日期分隔符UI，采用左侧竖条+底部细线的现代扁平设计，替换原有的圆角卡片设计。

## 最终设计方案

### 设计理念
- **扁平化**: 去掉圆角卡片和阴影效果
- **极简主义**: 使用简洁的竖条指示器
- **层次分明**: 通过颜色和字重区分今天和其他日期
- **视觉引导**: 左侧竖条形成时间轴的视觉连续性

### 核心元素

1. **左侧竖条指示器**
   - 尺寸: 4dp x 24dp
   - 今天: `primary` 颜色 (蓝色)
   - 其他日期: `outlineVariant` 颜色 (浅灰)
   - 圆角: `extraSmall` (轻微圆角)

2. **日期文本**
   - 字体: `titleMedium`
   - 今天: `Bold` 字重 + `primary` 颜色
   - 其他: `SemiBold` 字重 + `onSurfaceVariant` 颜色
   - 与竖条间距: 12dp

3. **底部分隔线**
   - 粗细: 1dp
   - 颜色: `outlineVariant` 半透明 (alpha = 0.5)
   - 作用: 视觉分隔，增强层次感

### 布局结构
```kotlin
Column {
    Row(vertical padding = 12dp) {
        Box(4dp x 24dp) // 竖条
        Text("今天 周三") // 日期文本
    }
    HorizontalDivider(1dp) // 底部细线
}
```

## 技术细节

### 修改的文件
`app/src/main/java/com/aura/football/presentation/home/components/DateSeparator.kt`

### 关键代码变更

**布局结构重构**:
- 从 `Box` + `Surface` → `Column` + `Row`
- 去掉 `Surface` 背景容器
- 使用纯文本 + 竖条指示器

**组件对比**:

| 元素 | 之前 (圆角卡片) | 现在 (竖条设计) |
|------|----------------|----------------|
| 容器 | Surface (带背景色) | Column (无背景) |
| 指示器 | 6dp圆点 | 4dp x 24dp竖条 |
| 字体 | titleSmall | titleMedium |
| 字重 | Bold/SemiBold | Bold/SemiBold |
| 圆角 | medium | extraSmall (仅竖条) |
| 阴影 | shadowElevation 2dp | 无 |
| 分隔线 | 无 | 底部1dp细线 |
| Padding | 20dp x 8dp (Surface内) | 12dp (Row垂直) |

**颜色方案**:

| 状态 | 竖条颜色 | 文字颜色 | 分隔线颜色 |
|------|---------|---------|-----------|
| 今天 | primary | primary | outlineVariant (50%) |
| 普通 | outlineVariant | onSurfaceVariant | outlineVariant (50%) |

### 代码实现

```kotlin
Column(background = background) {
    Row(padding = 12dp) {
        // 左侧竖条
        Box(
            modifier = Modifier
                .width(4.dp)
                .height(24.dp)
                .background(
                    color = if (isToday) primary else outlineVariant
                )
        )

        // 日期文本
        Text(
            text = text,
            style = titleMedium,
            fontWeight = if (isToday) Bold else SemiBold,
            color = if (isToday) primary else onSurfaceVariant
        )
    }

    // 底部分隔线
    HorizontalDivider(thickness = 1.dp, alpha = 0.5)
}
```

### 设计原则

- **扁平化设计**: 完全去掉立体效果(阴影、elevation)
- **极简主义**: 只保留必要的视觉元素
- **颜色语义**: 通过颜色传达信息(今天=蓝色，普通=灰色)
- **时间轴隐喻**: 左侧竖条形成视觉连续性，模拟时间线
- **呼吸感**: 底部细线提供视觉间隔，不拥挤
- **一致性**: 与Material Design 3扁平化趋势一致

### 设计优势

1. **更轻量**: 去掉背景容器，减少视觉重量
2. **更清晰**: 竖条指示器比圆点更醒目
3. **更现代**: 符合当前扁平化设计趋势
4. **更连贯**: 竖条形成时间轴的视觉连续性
5. **更专业**: 类似开发者工具的时间线UI风格
